# WhatsApp NF Automation

Automação desenvolvida em Python para envio automático de mensagens via WhatsApp Web utilizando Selenium e integração com planilhas Excel.

## Objetivo

Automatizar o envio de informações financeiras para motoristas/freteiros, reduzindo tempo operacional e eliminando tarefas manuais repetitivas.

## Tecnologias Utilizadas

* Python
* Selenium
* Pandas
* WebDriver Manager
* Excel Automation
* PyInstaller

## Funcionalidades

* Leitura automática de planilhas Excel
* Envio automático de mensagens personalizadas
* Integração com WhatsApp Web
* Logs de envio
* Tratamento de erros
* Delay aleatório entre mensagens
* Geração de executável (.exe)

## Estrutura do Projeto

```bash
src/
data/
logs/
```

## Como Executar

```bash
pip install -r requirements.txt
python main.py
```
